# Logo & Images

Drop your logo and image files in **`public/assets/images/Logo/`**.

## Recommended files

| File | Usage |
|------|--------|
| `logo.png` | Main logo (used in header & hero) |
| `logo-white.png` | White version for dark backgrounds |
| `logo-icon.png` | Icon only (no text) |

The app loads `logo.png` automatically. If missing, it falls back to the built-in SVG.

Supported formats: `.png`, `.svg`, `.webp`, `.jpg`
